# F-05 已上传文档列表

| 字段   | 值     |
|------|-------|
| 功能ID | F-05  |
| 模块   | 文档    |
| 优先级  | P1    |
| 版本   | V1.0  |
| 状态   | ✅ 已完成 |

---

## 1. 描述

返回所有已上传到向量库的文档文件名清单。

## 2. 用户故事

```
作为 [用户]，我希望 [查看已上传了哪些文档]，以便 [了解当前知识库的内容范围]。
```

## 3. 前置条件

无。

## 4. 后置条件

无状态变更。

## 5. 接口规范

| 元素 | 说明                                   |
|----|--------------------------------------|
| 方法 | `GET /api/documents`                 |
| 响应 | `AjaxResult<String[]>` — data 为字符串数组 |

### 响应数据字典

| 字段名  | 类型       | 必填 | 说明    | 示例值                    |
|------|----------|----|-------|------------------------|
| code | Integer  | 是  | 状态码   | 200                    |
| msg  | String   | 是  | 提示信息  | "success"              |
| data | String[] | 是  | 文件名数组 | `["手册.pdf","规范.docx"]` |

## 6. 异常/分支流程

| 场景  | 触发条件  | 处理方式  | 提示                       |
|-----|-------|-------|--------------------------|
| 无文档 | 向量库为空 | 返回空数组 | `{"code":200,"data":[]}` |
