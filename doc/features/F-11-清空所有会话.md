# F-11 清空所有会话

| 字段   | 值     |
|------|-------|
| 功能ID | F-11  |
| 模块   | 对话    |
| 优先级  | P2    |
| 版本   | V1.1  |
| 状态   | ✅ 已完成 |

---

## 1. 描述

一键删除所有活跃会话。

## 2. 用户故事

```
作为 [用户]，我希望 [一键清空所有会话]，以便 [重置整个对话环境]。
```

## 3. 前置条件

无。

## 4. 后置条件

| 变化   | 说明                                |
|------|-----------------------------------|
| 会话数据 | `ChatSessionManager` 清空所有会话消息和元数据 |
| 后续对话 | 所有 sessionId 被视为新会话               |

## 5. 接口规范

| 元素 | 说明                              |
|----|---------------------------------|
| 方法 | `POST /api/chat/sessions/clear` |
| 响应 | `AjaxResult`                    |

## 6. 异常/分支流程

| 场景    | 触发条件    | 处理方式      |
|-------|---------|-----------|
| 无任何会话 | 会话列表已为空 | 幂等操作，返回成功 |

## 7. 安全说明

清空操作**仅涉及当前应用实例的内存数据**（多实例部署下不清除其他实例的数据）。
